# Skills

This folder contains AI-readable skill cards for use in an AI app's Skills section.

## Available skill

### `vibe`

Path: `skills/vibe/SKILL.md`

Use this as the main Vibe Coding skill. It provides a governed workflow for coding, debugging, planning, testing, architecture review, and repository work.

## How to add this to an AI Skills section

1. Open your AI app or custom AI builder.
2. Go to the Skills section.
3. Create a new skill named `vibe` or `Vibe Coding`.
4. Copy the contents of `skills/vibe/SKILL.md`.
5. Paste it into the skill instruction/body field.
6. Save or publish the skill.

## Recommended trigger phrases

- `Use Vibe for this.`
- `Plan this with Vibe.`
- `Debug this using the Vibe workflow.`
- `Review this repo with Vibe.`
- `Turn this into a build plan using Vibe.`
